# 特性と技術仕様

{% hint style="info" %}
Difyでは、透明な製品特性と技術仕様ポリシーを採用しており、私たちの製品を全面的に理解した上で意思決定できるようにしています。この透明性は、技術選定に有益であるだけでなく、コミュニティメンバーが製品を深く理解し、積極的に貢献することを促進します。
{% endhint %}
